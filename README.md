# Next.js Login Page

This repository delivers a focused Next.js project that renders a responsive login screen with client-side validation and a mock authentication API.

## Getting Started

1. cd frontend
2. npm install
3. npm run dev

## Features

- Centered, accessible login card with email, password, and remember-me controls.
- Inline validation for required fields and basic email formatting.
- Mocked `/api/login` route with a simulated delay and hard-coded credentials.
- Disabled submit button while the request is running plus success/error messaging.
